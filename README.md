### Installation && Compile Instructions

Open your Command Line Interface (CLI) and run these commands. Once done go to
http://localhost:8080/

First Tab:
``` bash
$ cd Desktop/
$ git clone https://github.com/PerroneF5/GeekText.git
$ cd GeekText/
$ npm install
$ npm run dev
```

Second Tab:
``` bash
npm start
```
